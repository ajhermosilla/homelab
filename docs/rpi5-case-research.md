# Raspberry Pi 5 Case Research

3D printable case options for the RPi 5 Headscale server. Researched 2026-01-14.

## Requirements

- Fits Raspberry Pi 5
- Compatible with Active Cooler (thermal management for 24/7 operation)
- 3D printable locally in Paraguay
- Cool/geek/nerdy aesthetic
- Bonus: NVMe SSD support for future expansion

## Printing Tips

### Filament Choice

| Material | Use Case |
|----------|----------|
| **PETG** | Near active cooler (heat resistant) |
| **PLA** | Decorative parts, low-heat areas |
| **ABS** | Maximum heat resistance |

**Recommendation:** PETG for main body, PLA for accent pieces.

### Print Settings (General)

| Setting | Value |
|---------|-------|
| Layer height | 0.2mm |
| Infill | 20-30% |
| Supports | Check per model |
| Brim | Recommended for stability |

### Hardware Kit

Common parts needed across designs:

| Part | Quantity | Notes |
|------|----------|-------|
| M2.5 x 6mm screws | 4 | Mount Pi to case |
| M2.5 nuts | 4 | Some designs |
| M3 screws/nuts | Varies | Fan mounting |
| Rubber feet | 4 | Anti-slip |
