# Keychron K2C3 - Quickstart Guide

75% wireless mechanical keyboard for MacBook Air M1 and home workstation.

## Initial Setup

### Hardware Switches (Side Panel)

| Switch | Position | Purpose |
|--------|----------|---------|

| Cable/Off/BT | BT | Wireless mode |
| Mac/Win | Mac | macOS key layout |

### Pairing to MacBook

1. **Enter pairing mode:**Hold**Fn + 1** for 3-4 seconds
   - LED will flash rapidly (white/blue)

1. **On MacBook:** System Settings > Bluetooth
   - Wait for "Keychron K2" to appear
   - Click Connect

1. **Test:** Type something - should work immediately

## Multi-Device Pairing

The K2 remembers 3 Bluetooth devices:

| Shortcut | Action |
|----------|--------|

| Fn + 1 | Switch to device slot 1 (MacBook) |
| Fn + 2 | Switch to device slot 2 |
| Fn + 3 | Switch to device slot 3 |

#### To pair a new device to a slot

- Hold **Fn + [slot number]** for 3-4 seconds
- LED flashes = pairing mode
- Connect from the device

## Backlight Controls

| Shortcut | Action |
|----------|--------|

| Fn + Light bulb | Cycle brightness: High > Low > Off |
| Fn + < | Decrease RGB effect speed |
| Fn + > | Increase RGB effect speed |

## Function Keys (Mac Mode)

| Key | Mac Function |
|-----|--------------|

| F1 | Decrease brightness |
| F2 | Increase brightness |
| F3 | Mission Control |
| F4 | Launchpad |
| F5 | Decrease keyboard backlight |
| F6 | Increase keyboard backlight |
| F7 | Previous track |
| F8 | Play/Pause |
| F9 | Next track |
| F10 | Mute |
| F11 | Volume down |
| F12 | Volume up |

## Battery & Charging

- **Battery life:** ~72 hours (backlight off), ~10 hours (backlight on)
- **Charging:** USB-C port (right side)
- **Low battery:** LED flashes red
- **Wired mode:** Works while charging

## Troubleshooting

### Keyboard not connecting

1. Turn off Bluetooth, turn it back on
2. Hold Fn + 1 (3s) to re-enter pairing mode
3. Forget device on Mac, re-pair

### Keys not responding

- Check Cable/Off/BT switch is on BT
- Check battery (LED flashes red when low)
- Try wired mode to test

### Wrong key layout (@ symbol wrong, etc)

- Flip Mac/Win switch to Mac position
- macOS: System Settings > Keyboard > Identify keyboard (if prompted)

### macOS asking "Identify Keyboard"

Press the key to the **right** of left Shift (usually Z on ANSI layout).
