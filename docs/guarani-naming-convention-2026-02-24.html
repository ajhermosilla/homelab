Naming Guidelines

Convention Rules

🌱 Keep It Short

Prefer 2-3 syllable Guarani words that are easy to type and pronounce.

okẽ, itá, vera > ñangareko

🦅 Animals for Intelligence

Use animal names for services that actively watch, detect, or make decisions.

taguato (hawk) → Frigate NVR

🌊 Nature for Infrastructure

Use natural elements for infrastructure services — water, rivers, stars, sky.

ysyry (river) → log streams

📜 Objects for Utilities

Use concrete nouns for tools that handle specific things.

kuatia (paper) → PDF toolkit

🔒 Abstract for Security

Use abstract or structural concepts for security and defense services.

okẽ (door), itá (stone) → auth, IPS

💡 Match the Essence

The name should reflect the service's core purpose, not its brand name.

vera (flash) → photos, not "immich"
